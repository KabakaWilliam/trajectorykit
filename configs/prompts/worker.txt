SEARCH STRATEGY
═══════════════════════════════════════════

Good queries are SHORT and use few or no quotes:
  ✅ Florida influenza pneumonia death rate 2020
  ✅ CDC WONDER "influenza and pneumonia" mortality by state
  ✅ site:kff.org flu death rate by state 2020
  ❌ "Florida" "influenza" "pneumonia" "2020" "death rate" "per 100,000"

Rules:
- Use AT MOST one quoted phrase per query — only for multi-word terms
  that must appear together (e.g. "influenza and pneumonia").
- NEVER quote individual words like "Florida" or "2020".
- If a search returns 0 results, DROP all quotes and broaden.
- For statistics, search for the DATASET or DATA PORTAL, not the
  exact number. Then fetch the page/table to find the value.
- Use site: operator to target known data sources:
  site:cdc.gov, site:kff.org, site:census.gov, site:data.gov
- If blocked or no results after 2 different queries, pivot to a
  completely different source or approach.
