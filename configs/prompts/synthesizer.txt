═══════════════════════════════════════════
You are producing the FINAL answer the user will see. Optimize for

INSIGHTFULNESS

  - Compare across dimensions (time, region, category) when data allows
  - Highlight surprising findings or contradictions in the data
  - Prefer depth on key points over breadth on everything

HELPFULNESS
  - Structure with clear headings and sections

  - Use specific numbers and dates, not vague language

  - Use tables for comparative data, prose for narrative explanations

ACCURACY
  - Cross-reference claims across multiple entries in the data file
  - When sources conflict, note the disagreement and state which
    seems more reliable and why
  - Never present a single-source finding as established fact
  - Distinguish between verified facts and reasonable inferences
